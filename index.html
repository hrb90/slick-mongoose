You can draw a planar graph on the canvas. Draw a new vertex by clicking an empty part of the graph; draw an edge by clicking two existing vertices. Try to keep all the edges connected.

If you draw a graph and then click the "Color" button, it will (likely) add some extra, blue edges to the graph, and then color the graph with five colors so no vertices connected by an edge have the same color. If there is a graph that the page won't color, or the graph is miscolored, please open a new issue :). Thanks again!
