You can draw a planar graph on the canvas. Draw a new vertex by clicking an empty part of the graph; draw an edge by clicking two existing vertices. Try to keep all the edges connected.

If you draw a graph and then click the "triangulate" button, it will add blue edges to the graph so that all inner faces are triangles. If there are faces that aren't triangles, or there are crossing edges, please let me know in this issue. Please include the log of how your graph was constructed (open up the developer tools console and type "graphLog").

If you run into any other problems, please open a new issue :). Thanks again!
