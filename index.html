You can draw a planar graph on the canvas. Draw a new vertex by clicking an empty part of the graph; draw an edge by clicking two existing vertices. Try to keep all the edges connected.

If you draw a graph and then click the "Color" button, it will (likely) add some extra, blue edges to the graph, and then color the graph with five colors so no vertices connected by an edge have the same color.

If the graph is miscolored (i.e. two vertices connected by an edge are the same color) please add a comment to this issue. Please include the output of getAnimString() in the console.

If there is a graph that the page won't color, please open a new issue and include the exception in the console, if any, and preferably a screenshot of the graph. Thanks again! :)
